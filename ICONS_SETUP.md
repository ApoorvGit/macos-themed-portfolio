# macOS Icons Setup

## How to Add Real macOS Icons

1. Visit: https://www.figma.com/community/file/1216394183263141647/all-macos-big-sur-icons-70-icons

2. Click "Open in Figma" (you may need to create a free Figma account)

3. Export the following icons as PNG or SVG (512x512 or larger):
   - **Finder** → Save as `finder.png` for Projects
   - **Briefcase/Work** → Save as `briefcase.png` for Experience  
   - **Terminal** → Save as `terminal.png` for Skills
   - **Safari** → Save as `safari.png` for Blog
   - **Photos** → Save as `photos.png` for Gallery
   - **Mail** → Save as `mail.png` for Contact

4. Save all icons to: `/src/assets/icons/`

5. The code will automatically use these icons once they're in place.

## Alternative: Download from macosicons.com

Visit https://macosicons.com/ and download:
- Finder
- Safari
- Terminal
- Photos
- Mail
- Any briefcase/work icon

Save them to `/src/assets/icons/` with the names above.
